# Customer Intelligence Platform

An end-to-end data science project that analyzes e-commerce customer behavior using the Olist Brazilian E-commerce dataset. The project aims to transform raw transactional data into actionable business insights through data engineering, machine learning, visualization, and deployment.

Current Status
🟢 Phase 1: Business Understanding

## Dataset

This project uses the Olist Brazilian E-commerce Public Dataset.

1. Download the dataset from Kaggle.
2. Extract the files.
3. Place them in:

data/raw/olist-ecommerce-public-dataset/
